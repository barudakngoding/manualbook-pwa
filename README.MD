## ğŸš€ Cara Install PWA

### Di Android:
1. Buka website di Chrome
2. Tap menu (3 titik)
3. Pilih "Add to Home screen"
4. Tap "Add"

### Di iOS:
1. Buka website di Safari
2. Tap tombol Share
3. Pilih "Add to Home Screen"
4. Tap "Add"

### Di Desktop:
1. Buka website di Chrome
2. Klik icon install di address bar
3. Klik "Install"
